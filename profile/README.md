# Evolution X

Evolution X is a custom Android ROM that focuses on replicating the Google Pixel experience, with added customization. Based on [LineageOS](https://github.com/LineageOS).

- Pixel exclusive features (Unlimited Photos storage, Pixel Wallpapers, Pixel lock screen clock styles, Pixel sound effects etc.)
- Additional customization (LMO Freeform/Sidebar, App lock, etc.)
- Frequent releases
- Community technical support
- Support for numerous Android devices
